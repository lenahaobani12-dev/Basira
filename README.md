## 📝 الوصف
**بصيرة** هو مساعد قانوني ذكي مصمم خصيصًا لذوي الإعاقة المزدوجة (الكفيف الأبكم) ليمكنهم من قراءة وفهم النصوص القانونية والعقود باستقلالية تامة، عبر تحويل المستندات الورقية إلى صوت مسموع وتنبيهات حسية.

## 🚀 التشغيل (Installation & Setup)
المشروع مصمم ليعمل في بيئة **Google Colab**. اتبع الخطوات التالية للتشغيل:

1. **تثبيت المتطلبات:** قم بتشغيل الأوامر التالية في خلية الكود الأولى:
```bash
# تثبيت محرك التعرف على النصوص للغة العربية
!apt-get install tesseract-ocr -y
!apt-get install tesseract-ocr-ara -y

# تثبيت المكتبات البرمجية اللازمة
!pip install pytesseract opencv-python gTTS openai
```

2. **الإعداد:** ضع مفتاح الوصول (API Key) الخاص بنماذج علم في قسم الإعداد:
```python
client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://elmodels.ngrok.app/v1"
)
```

3. **التنفيذ:** * قم بتشغيل كود الكاميرا والضغط على زر **"📷 التقاط صورة للتحليل"**.
* سيقوم النظام تلقائياً بمعالجة الصورة وإصدار تقرير صوتي واهتزازات تنبيهية.

## 🛠️ التقنيات (Tech Stack)
- **Framework:** Python (Google Colab Environment).
- **AI Models:** - `nuha-2.0`: لتحليل محتوى الصور وتصنيف مستوى الحالة (آمن/خطر).
    - `elm-tts`: لتحويل النتائج القانونية إلى صوت طبيعي.
- **Computer Vision:** - `Tesseract OCR`: لاستخراج النصوص العربية والإنجليزية.
    - `OpenCV`: لمعالجة وتحسين الصور الملتقطة.
- **Haptic Feedback:** نظام اهتزاز برمجي عبر JavaScript لتنبيه المستخدم حسياً.
