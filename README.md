# Basira

## 🚀 فكرة المشروع

"بصيرة" هو نظام ذكي يعتمد على الذكاء الاصطناعي لمساعدة ذوي الإعاقة البصرية والسمعية على فهم المستندات القانونية والتفاعل معها بشكل مستقل.



## 🎯 المشكلة

يواجه المستخدمون من ذوي الإعاقة:

* صعوبة في قراءة العقود والمستندات
* الاعتماد على الآخرين مما يقلل الخصوصية
* عدم فهم المصطلحات القانونية المعقدة



## 💡 الحل

يوفر "بصيرة":

* 📷 قراءة المستندات عبر الكاميرا (OCR)
* 🔊 تحويل النص إلى صوت (TTS)
* 📳 نظام توجيه وتنبيه (Haptic Feedback)
* 🧠 تبسيط المعلومات القانونية



## ⚙️ آلية العمل

1. تصوير المستند باستخدام الكاميرا
2. استخراج النص باستخدام OCR
3. تحويل النص إلى صوت باستخدام TTS
4. توجيه المستخدم عبر إشارات حسية (اهتزازات)



## 📳 نظام التنبيهات

* اهتزاز عند اكتشاف المستند
* تنبيه عند الاقتراب أو الابتعاد
* إشعارات صوتية مثل:
  "تم التقاط المستند"
  "يوجد خطر، انتبه"



## 🛠️ التقنيات المستخدمة

* Python
* OpenCV
* Tesseract OCR
* Text-to-Speech (gTTS)
* Web Vibration API

طريقة التشغيل:

لتشغيل النظام، يتم فتح المشروع على Google Colab، ثم رفع الملفات المطلوبة (Upload) وتشغيل الكود (Run). بعد تشغيله، يتم فتح واجهة الكاميرا تلقائياً، ويمكن للمستخدم التقاط الصورة مباشرة بالضغط على زر الالتقاط، ليتم بعدها تحليل الصورة ومعالجتها فوراً
